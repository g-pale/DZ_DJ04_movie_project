# Movie Project

Веб-приложение на Django для управления базой данных фильмов с возможностью добавления и просмотра информации о фильмах, их описаний и отзывов.

## Описание проекта

Проект представляет собой простое веб-приложение для работы с базой данных фильмов. Пользователи могут:
- Добавлять новые фильмы с названием, описанием и отзывом
- Просматривать список всех добавленных фильмов
- Видеть подробную информацию о каждом фильме

## Технологии

- **Python 3.x**
- **Django 6.0**
- **SQLite** (база данных)
- **Bootstrap 5** (стилизация интерфейса)
- **Bootstrap Icons** (иконки)

## Модель данных

### Film (Фильм)

Модель содержит следующие поля:

- **title** (CharField) - Название фильма (максимум 200 символов)
- **description** (TextField) - Описание фильма
- **review** (TextField) - Отзыв о фильме
- **created_at** (DateTimeField) - Дата и время создания записи (автоматически)

## Установка и запуск

### Требования

- Python 3.8 или выше
- Django 6.0 или выше

### Шаги установки

1. **Клонируйте репозиторий или перейдите в директорию проекта:**
   ```bash
   cd movie_project
   ```

2. **Создайте виртуальное окружение (рекомендуется):**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Для Linux/Mac
   # или
   venv\Scripts\activate  # Для Windows
   ```

3. **Установите зависимости:**
   ```bash
   pip install django
   ```

4. **Примените миграции базы данных:**
   ```bash
   python3 manage.py makemigrations
   python3 manage.py migrate
   ```

5. **Создайте суперпользователя для доступа к админ-панели (опционально):**
   ```bash
   python3 manage.py createsuperuser
   ```

6. **Запустите сервер разработки:**
   ```bash
   python3 manage.py runserver
   ```

7. **Откройте браузер и перейдите по адресу:**
   ```
   http://127.0.0.1:8000/
   ```

## Использование

### Основные страницы

1. **Главная страница** (`/`)
   - Автоматически перенаправляет на страницу со списком фильмов

2. **Список фильмов** (`/films/`)
   - Отображает все добавленные фильмы в виде карточек
   - Каждая карточка показывает название, дату добавления, краткое описание и отзыв
   - Кнопка "Подробнее" открывает модальное окно с полной информацией
   - Кнопка "Добавить фильм" ведет на страницу добавления

3. **Добавление фильма** (`/films/add/`)
   - Форма с тремя полями:
     - Название фильма (обязательное)
     - Описание фильма (обязательное)
     - Отзыв (обязательное)
   - После успешного добавления происходит перенаправление на страницу со списком фильмов

4. **Админ-панель** (`/admin/`)
   - Доступна после создания суперпользователя
   - Позволяет управлять фильмами через интерфейс Django

## Особенности интерфейса

- **Адаптивный дизайн** - сайт корректно отображается на всех устройствах
- **Bootstrap 5** - современный и красивый интерфейс
- **Иконки** - использование Bootstrap Icons для улучшения визуального восприятия
- **Модальные окна** - просмотр полной информации о фильме без перезагрузки страницы
- **Навигационное меню** - удобная навигация между страницами
